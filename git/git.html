• git init: Creates a .git folder and makes a repo (only do this one time)
  • git add . : stages files to be committed
  • git status: Show the diff between last commit and now
  • git commit -m: Takes the staged state and saves it
  • git push (origin master): this tells git to push changes to remote repo

Safe commands to run:

  • git status: show diff between last commit and now
  • git help: shows the overview of git commands
  • git log -b: show all commit messages
